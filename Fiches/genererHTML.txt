TITRE

	Générer une page HTML

DESCRIPTION

	Permet de générer une page HTML facilement afin qu'un lecteur puisse jouer en ligne sur son navigateur internet

HYPOTHESES

	L'acteur a un navigateur internet installé sur son ordinateur qui lui permet de lire le code source d'une page HTML + ouvrir une page HTML de manière classique.
	Le système sait identifier un message d'erreur d'un navigateur internet (pour une mauvaise génération du code source)

ACTEURS

	L'auteur du livre

SCENARIO NOMINAL

	 1. L'acteur sélectionne Générer Version HTML
	 2. Le sytème analyse chaque section du livre
	 3. Le système ouvre le navigateur choisi dans les options du logiciel et affiche le code source HTML de chaque section dans un onglet différent
	 4. Le système vérifie que le navigateur n'a pas renvoyé de message d'erreur suite à l'ouverture de la page HTML
	
	[3- 4] Le système se sert d'un parser XML pour vérifier que le code
html généré est correct.
	
	 5. Le système affiche une nouvelle fenêtre : Génération terminée avec succès

POST-CONDITION

	L'auteur sait que ses pages HTML sont jouables par un futur lecteur depuis son navigateur web personnel


SCENARIO ALTERNATIF

	 1. Le texte de l'auteur est vide

	 1.1 A la fin du SN2, le système signale que le texte est vide dans une nouvelle fenêtre
	 1.2 Le système ouvre le navigateur et affiche le code source HTML d'une page vierge
	 1.3 Retour au SN4

SCENARIO EXCEPTION

	 1. La génération HTML s'est mal passée
	 
	 1.1 Au SN3, le navigateur renvoie un message d'erreur du code source HTML d'une des pages générées
	 1.2 Le système ferme l'onglet de la page HTML mal générée
	 1.3 Le système ouvre une nouvelle fenêtre d'erreur et signale le numéro de la section qui a mal été générée
